메모장 프로젝트에서는 언제든지 자신만의 메모를 기록할 수 있는 메모장을 구현했습니다. 
Spring과 MySQL을 사용하여 HTTP 데이터를 객체로 처리하고, JPA를 기반으로 CRUD 기능을 구현하며 Auditing을 적용했습니다. IoC와 DI 개념을 적용하여 역할을 분리하고, 3 Layer Architecture를 도입하여 코드의 유지보수성을 높였습니다. 

기능

- 메모 작성하기
- 메모 조회하기
- 메모 수정하기
- 메모 삭제하기
